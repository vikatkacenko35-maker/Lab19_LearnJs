## Краткое описание работы

В ходе лабораторной работы были изучены следующие темы языка **JavaScript**:

- **Циклы**: `for`, `while`, `do...while`, операторы `break` и `continue`, вложенные циклы
- **Функции**: объявление, параметры по умолчанию, функции как значения, замыкания (closure), стрелочные функции (arrow functions)
- **Массивы**: создание, доступ к элементам, длина массива, методы `push()`/`pop()`, перебор (`for`, `for...of`), поиск элементов (`indexOf()`, `includes()`)
- **Объекты**: создание литералом, доступ к свойствам, добавление/удаление свойств, методы объектов, перебор свойств (`for...in`)

Также проведено сравнение синтаксиса и возможностей JavaScript с C#.
